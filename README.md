# Indian cricketers-image-classification


# How to run ?

### STEP 01- Clone the repository


```bash
project repo : https://github.com/nisha19d/cricketers-image-classification
```

### STEP 02- Create a conda environment after opening the repository

```bash
conda create -n imageclasify python=3.10 -y
```

```bash
conda activate imageclasify
```

### STEP 03- Run the following commands in conda environment


```bash
cd server
```

```bash
python server.py
```

Now,
```bash
open up localhost:5050
```
